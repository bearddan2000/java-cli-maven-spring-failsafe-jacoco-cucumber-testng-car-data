# java-cli-maven-spring-failsafe-jacoco-cucumber-testng-car-data

## Description
A POC for spring app using testng
and cucumber framework with jacoco, failsafe and
surefire plugins.

## Tech stack
- java
- maven
  - spring
  - testng
  - jacoco
  - failsafe
  - surefire
  - cucumber

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
